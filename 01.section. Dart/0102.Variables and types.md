# Phase 1-2: 변수와 타입

## 1. 개념 및 정의

### 1. 변수 선언 키워드 (4가지)
Dart는 변수 선언 방식이 Python보다 엄격하지만, 타입 추론이 강력해서 실제로는 간결함.
 
| 키워드 | 설명 | 재할당 | 컴파일 타임 상수 | 사용 예 |
|--------|------|--------|-----------------|--------|
| `var` | 타입 자동 추론 (선언 시 결정) | ✅ | ❌ | `var word = 'apple';` |
| `final` | 한 번만 할당 (런타임 상수) | ❌ | ❌ | `final now = DateTime.now();` |
| `const` | 컴파일 타임 상수 | ❌ | ✅ | `const maxCards = 100;` |
| `dynamic` | 런타임에 타입 변경 가능 | ✅ | ❌ | `dynamic x = 1; x = 'hi';` (지양) |


### 2. `var` vs. 타입 명시

```dart
// 두 방식은 완전히 동일하다
var word = 'apple';     // String으로 추론
String word = 'apple';  // 명시적 선언
```

- 언제 `var`를 쓸까?
> 우변의 타입이 명백할 때(`var count = 0;`)
> 우변이 길어서 좌변에 타입을 또 쓰면 중복일 때(`var card = MyCard.create(...)`)

- 언제 타입을 명시할까?
> 함수 시그니처 (가독성)
> 클래스 필드 (계약)
> 추론 결과가 직관적이지 않을 때


### 3. `final` vs. `const` (핵심 차이)

```
┌─────────────────────────────────────────────┐
│  final = 런타임 상수                         │
│  ─────────────────────                      │
│  실행 중에 값이 결정되어도 OK                 │
│  예: final now = DateTime.now();            │
├─────────────────────────────────────────────┤
│  const = 컴파일 타임 상수                    │
│  ─────────────────────                      │
│  컴파일 시점에 이미 값이 정해져 있어야 함      │
│  예: const pi = 3.14159;                    │
│  예: const DateTime.now();  ❌ 에러         │
└─────────────────────────────────────────────┘
```


### 4. `dynamic`은 왜 지양해야 하나?

```dart
dynamic x = 5;
x = 'hello';        // ✅ 컴파일 에러 없음
x.foo();            // ✅ 컴파일 에러 없음 (실제 실행 시 런타임 에러)
```

`dynamic`은 타입 검사를 완전히 무력화함. Dart의 강점인 타입 안전을 포기하는 셈.
외부 JSON 파싱 같은 특수한 경우에만 사용

> 🔍 `Object`와의 차이:
> `Object`는 타입 검사를 하므로 `obj.foo()` 호출 시 컴파일 에러. `dynamic`은 검사를 건너뜀.


### 5. 기본 자료형

| 타입 | 설명 | 범위/예시 |
|------|------|-----------|
| `int` | 정수 | 64비트 정수 (2의 63제곱 범위) |
| `double` | 부동소수점 | 64비트 IEEE 754 |
| `num` | int + double의 부모 | `num x = 1; x = 1.5;` 모두 OK |
| `String` | 문자열 (UTF-16) | `'apple'`, `"banana"` |
| `bool` | 참/거짓 | `true`, `false` (1/0 안 됨) |
| `Symbol` | 식별자 심볼 | `#mySymbol` (거의 안 씀) |
| `Runes` | 유니코드 정수 시퀀스 | 이모지 다룰 때 사용 |


### 6. 형 변환 (Type Conversion)

| 변환 | 메서드 | 예시 |
|------|--------|------|
| String → int | `int.parse('5')` | `int.parse('42')` → `42` |
| String → double | `double.parse('3.14')` | `3.14` |
| 숫자 → String | `.toString()` | `42.toString()` → `'42'` |
| int → double | `.toDouble()` | `5.toDouble()` → `5.0` |
| double → int | `.toInt()` / `.round()` | `3.7.toInt()` → `3`, `3.7.round()` → `4` |
| 안전한 파싱 | `int.tryParse('abc')` | 실패 시 `null` 반환 |


### 7. 문자열 다루기

```dart

String b = "World";                     // 큰따옴표도 가능 (동일)

String f = 'a' 'b' 'c';                 // 인접 문자열 자동 연결: 'abc'
 
// 보간

print('count: $n');                     // 변수
print('next: ${n + 1}');                // 표현식
print('len: ${'hello'.length}');        // 메서드 호출
```
